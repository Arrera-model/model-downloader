# model-downloader

## Paramètre d’entrée du constructeur

L'objet ne prend aucun paramètre. 

## Méthodes

- **get_model_list** : Permet de voir tous les modèles qui sont téléchargeables.
- **get_data_model** : Permet de récupérer toutes les données du modèle.
- **get_model_download** : Voir les modèles téléchargés sur l'ordinateur.
- **download_model** : Permet de télécharger un modèle sur l'ordinateur
- **get_path_model** : Permet de récupérer le chemin d'un modèle téléchargé.
